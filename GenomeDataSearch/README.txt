Please follow the steps to test the algorithm 
1- Create a data folder in the project folder and copy probes.txt there.
2- Run GenomeQueryTests which already has a set of tests. You can see info lines in the console processing each chromosome.
  - You can add your own queries if you follow the format in the javadoc of the file
3- Check the output text files files created per query in the ./results folder of the project.
